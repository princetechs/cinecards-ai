# Content Licensing

CineCards AI uses a split licensing model so code and educational content can be reused with the right expectations.

- **Code** (everything under `components/`, `lib/`, `src/`, `types/`, build config): licensed under **Apache-2.0**. See `LICENSE` at the repo root.
- **Content** (everything in `data/`, term text, diagrams, prompt templates, and other learning assets): licensed under **Creative Commons Attribution 4.0 International (CC BY 4.0)**. See `content/LICENSE-CONTENT` in this folder.

## What this means for reuse

You may copy, adapt, and redistribute the content for any purpose, including commercially, as long as you give appropriate credit and link to the license.
